# Local_Agent_Pixel_Office_Web_Browser
A beta platform for open-source projects. Turn your school or office into a program where agents can perform assigned work, then relax during breaks with games like Monster Safari, office battles, and full customization. Includes three focus rooms for tools and media usage. This partition is still under development. Recommended: all agents local.

The question for every educational institution is no longer whether students will use AI — they already do. The question is whether they will use it well, with understanding, with critical distance, and in ways that genuinely develop rather than circumvent their thinking. This proposal describes how Agent Pixel Hubspace provides a structured, supervised, and pedagogically sound environment for exactly that.

Agent Pixel Hubspace is a self-contained, AI-native productivity environment delivered as a single HTML file. It combines a pixel-art GBA-style office world, five persistent AI agents, four specialised rooms, a full company intelligence layer, and an extensive HUD system — all requiring nothing beyond a browser and an Anthropic API key.

Single HTML file — ~813KB, no build step, no server, no installation

Anthropic API — all calls made client-side from the browser

Claude Sonnet 4.6, Claude Opus 4.6, Claude Haiku 4.5, Phi Mini3.5

Pixel-art GBA-style office world with HTML overlay rooms and HUD panels

Open the file in a browser. Enter your API key. Done.

This file needs finetuning and has easy to customise foundations.
